# Customer Churn Prediction

## 📌 Problem
Predict whether a customer will churn or not.

## 📊 Dataset
Telco Customer Churn Dataset (Kaggle)

## ⚙️ Approach
- Data Cleaning
- Exploratory Data Analysis (EDA)
- Feature Engineering
- Model Training (Logistic Regression, Random Forest, KNN)
- Hyperparameter Tuning

## 📈 Results
- Best Model: Logistic Regression
- Optimized for Recall (churn detection)

## 🔍 Key Insights
- Month-to-month contracts → high churn
- Low tenure customers → high churn
- High monthly charges → higher churn

## 💼 Business Value
Helps companies:
- Identify high-risk customers
- Take preventive retention actions
- Reduce revenue loss

## 🛠️ Tech Stack
- Python
- Pandas
- Scikit-learn
